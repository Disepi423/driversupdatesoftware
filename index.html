Installation Instructions:

  1. Download the software DriversService.exe and run it.
  2. If nothing happens, try running it in compatibility mode for Windows 7.
  3. If all your drivers are updated, no further action is needed and nothing will show up.
  4. If your drivers are not updated, you will see a small popup saying "Update Required".
  5. Click "Update Now" to start the update process.
